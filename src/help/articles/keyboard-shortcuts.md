# Keyboard shortcuts

Tickd is built to be operable from the keyboard once you know a few.

## Global

| Shortcut | Action |
|----------|--------|
| `/` | Focus the quick-entry box (top of sidebar) |
| `⌘K` / `Ctrl K` | Open the search palette |
| `?` | Show this shortcut list (works on most surfaces) |
| `Esc` | Close any open modal or popover |
| `g t` | Jump to **Today** view |
| `g l` | Jump to **List** view |
| `g r` | Jump to G**r**id view |
| `g p` | Jump to **PIC** view |
| `g c` | Jump to **Calendar** view |
| `g d` | Jump to **Docs** view |
| `g i` | Jump to **Inbox** |

> [!TIP]
> The `g` shortcuts are chord-style — press `g`, then within ~1 second press the next letter. Standard gmail / linear pattern.

## Quick entry

| Shortcut | Action |
|----------|--------|
| `Enter` | Save task |
| `@FirstName` | Assign PIC inline (autocompletes) |
| `#tag` | Add a tag |
| `due Mon` / `due Friday` / `due tomorrow` | Set due date inline |
| `high` / `low` | Set priority |

## Task modal

| Shortcut | Action |
|----------|--------|
| `n` | Jump to the Comments tab |
| `a` | Jump to the Activity tab |
| `⌘Enter` / `Ctrl Enter` | Save and close |
| `⌘S` / `Ctrl S` | Save without closing |
| `Esc` | Close (autosaves pending changes first) |

## Doc editor

| Shortcut | Action |
|----------|--------|
| `⌘B` / `Ctrl B` | Wrap selection in **bold** |
| `⌘I` / `Ctrl I` | Wrap selection in *italic* |
| `⌘E` / `Ctrl E` | Wrap selection in `code` |
| `⌘S` / `Ctrl S` | Force-flush save |

## List view

| Shortcut | Action |
|----------|--------|
| `j` / `k` | Move focus down / up between rows |
| `Enter` / `o` | Open the focused task |
| `x` | Toggle selection (for bulk actions) |

## Comments

| Shortcut | Action |
|----------|--------|
| `⌘Enter` / `Ctrl Enter` | Post comment |
| `@` | Open @mention autocomplete |
| `Esc` | Cancel reply |
